Marketing Campaign ROI Analysis Dashboard
This dashboard is designed to help marketers and data analysts assess the Return on Investment (ROI) of various marketing campaigns. It provides a clear and concise view of campaign performance, enabling users to track key metrics such as total costs, revenue generated, and the overall effectiveness of each campaign. With easy-to-navigate visualizations, this tool allows businesses to make data-driven decisions, optimizing future marketing strategies based on historical performance.

Built using Excel/Power BI, this ROI dashboard includes features such as dynamic filtering, drill-down capabilities, and customizable charts for in-depth analysis. The dataset consists of over 10,000 rows, simulating real-world campaign data to deliver robust insights. This project demonstrates key data analytics skills, including data cleaning, visualization, and reporting, aimed at maximizing business impact.
